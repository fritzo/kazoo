\abstract{
 \kazoo is a trainable real-time audio transformation system
 for use in sound and music synthesis.
 
}

\section{Introduction}

Electronic musical instrument expressivity is often limited by input bandwidth.
The \kazoo system addresses this by using sound as the input medium.

We develop a novel extensions of deep restricted boltzmann machines
(deep belief nets) for time-series data.

The kazoo transformation consists of
  (1) a psychoacoustic front-end,
  (2) an adaptive front-end,
  (3) a deep dynamical model,
  (4) an adaptive back-end mirroring the front-end, and
  (5) a psychoacoustic back-end mirroring the front end.
  Components (2-4) constitute a deep transition belief net.
  Levels (2) and (4) use the same probabilistic models.
  }
